# Pokedex-WebApp

A web-based Pokédex application that allows users to search and view Pokémon details like stats and images.

## 🚀 Features
- Search Pokémon by name  
- View basic stats  
- Display Pokémon images  

## 🛠️ Tech Stack
- Java (JSP & Servlets)
- HTML, CSS
- PokéAPI

## ⚙️ How to Run
1. Clone the repository
2. Open in Eclipse/IntelliJ
3. Configure Apache Tomcat
4. Run the project
5. Open in browser: http://localhost:8080/

## 🌐 API
Data fetched from PokéAPI
